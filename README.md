# chat-app

## Description
    Chat Application created using reactJS & Firebase
## Features
    Encrypted Real-time Chats
    Send Images & stickers
    Cloud Syncing
    Combines Phone no. & Email contacts
## Technologies Used
    ReactJS
    Vite
    Firebase
    SCSS (Sass)
## How to run
    git clone {repo-link}
    cd into the repo
    npm install
    npm run dev
    Open http://localhost:5173 to view it in your browser.
